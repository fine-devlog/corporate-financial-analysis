## 概要
システムを利用してもらう上での注意事項の表示方法や書式の案をまとめた。
## 目的
企業データを扱う以上、就活生や投資家の方が本ツールの利用で何らかの損益を受ける可能性がある。
しかし、企業の業績というのは刻一刻と変わるもので、絶対ということは存在しない。それを理解していただき、適切に利用していただくための注意書きが必要であると考えた。
注意書きでもさまざまな形式があるため、数パターンを作成して最も私の開発するプログラムの雰囲気に沿ったものを選ぼうと考えている。
