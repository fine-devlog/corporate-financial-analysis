# corporate-financial-analysis
企業の財務データ分析ツール

## このリポジトリについて
企業の財務データ分析ツールに関連したプログラムやアイデア、アンケートプログラムなどを掲載していく。
すでにこのアカウントで作成した「〇〇-old」というリポジトリの内容については、すべてこのリポジトリに集約予定。

## 各フォルダの概要
- 「dashbosrd_app」：完成した分析プログラムを掲載予定。
- 「design-notes」：ツールに関する設計アイデアをまとめている。
- 「survey_app」：ヒアリングを行うためのアンケートツールを掲載予定。
- 「disclaimer_options」：注意書きのサンプルコードをまとめている。
- 「validation」：バリデーション（異常値検出）のサンプルコードを掲載予定。
- 「analysis-sandbox」：ツールの開発過程で作成した試作品や、機能ごとに作成したプログラムを掲載予定。
